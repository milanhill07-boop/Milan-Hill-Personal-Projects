# Milan Hill's Code Space

SciFi_Movie_Recs:

In this program, targeting an audience interested in finding the right science fiction movie to watch, users receive a randomly selected recommendation from a curated collection of 5-star-rated films. The program provides an in-depth review and displays the movie's cover image to help users discover their next favorite sci-fi film

Tetris: 

This program displays Tetris high scores and basic stats. Users can view the highest, lowest, and average scores, as well as enter a new score to update the records.
